    This class implements a variation of Stochastic Universal Sampling (SUS) that we call Biased Shifted Stochastic Universal Sampling (Biased Shifted SUS), which uses shifted fitness values and integrates the use of a bias function with SUS to enable transforming the shifted fitness values prior to the stochastic selection decisions.
